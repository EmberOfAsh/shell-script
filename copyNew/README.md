# 复制新文件
复制指定目录下，新产生的文件/文件夹到新的目录

## 工作原理
读取文件/文件夹最后修改时间，和指定时间做比较，如果较新，则复制到新保存的目录下

## 示例
```Linux
./copyNew.sh /mnt/d/ping /mnt/d/test 2018-12-28 
```

## 参数
| 参数        | 解释    |  例子  |
| --------   | -----:   | :----: |
| param1| 需要比较的目录    |   /mnt/d/ping    |
| param2| 比较结果保存目录  |   /mnt/d/test    |
| param3| 比较时间，时间格式必须是 yyyy-mm-dd   |   2018-12-28    |
